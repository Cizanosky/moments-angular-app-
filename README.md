# Moments

Moments is a Single Page Application (SPA) developed with Angular, inspired by a simplified social media platform.

The project was built to practice and apply core Angular concepts, including component architecture, parent-child communication, event binding, and dynamic rendering.

---

## Project Overview

The application allows users to create and display posts ("moments") containing:

- An image
- A title
- A description
- Comments

The system simulates a social media feed interface where moments are displayed in a structured and interactive layout.

Although the project is not connected to a real authentication system or production database, it demonstrates frontend architecture principles and structured component interaction.

---

## Key Angular Concepts Applied

- Component-based architecture
- Parent-to-child and child-to-parent communication (@Input / @Output)
- Event binding
- Property binding
- Conditional rendering
- Dynamic lists with *ngFor
- Form handling
- HTTP communication (if applicable)
- Service layer for data management
